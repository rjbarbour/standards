# Manifest and schema reference

This page is a reference companion to the site's README.llm vNext example
implementation.

## Canonical artifacts

- [`/README.llm`](/README.llm)
- [`/.well-known/llm-index.json`](/.well-known/llm-index.json)
- [`/schemas/readme-llm-vnext/manifest-0.1.schema.json`](/schemas/readme-llm-vnext/manifest-0.1.schema.json)

## What each artifact does

- `README.llm` is the base machine entry point.
- `llm-index.json` is the retrieval manifest.
- the schema file defines the manifest structure.

## When to use this page

Use this page when you need an exact route, field name, or canonical schema
URL. Use the how-to or tutorial sections when you need workflow guidance
instead of factual lookup.
